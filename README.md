Project 1: Cloud Job Market Analysis

In this project I looked at a dataset of cloud-related job roles to see what kind of demand exists for different cloud platforms.

The goal was to explore the data and see patterns in the job market for technologies like AWS, Azure, and Google Cloud.

Things I explored in this project:
	•	Comparing demand for AWS, Azure, and Google Cloud roles
	•	Looking at trends in cloud job listings
	•	Creating charts to visualize the data

Tools used:
	•	Python
	•	Pandas
	•	Matplotlib

Files in this project:
	•	project.py
	•	cloud_jobs.csv

Project 2: Student Performance Analysis

This project analyzes how study time and attendance can affect student grades.

I used a small dataset of students to explore relationships between study hours, attendance, and final grades.

Things I explored in this project:
	•	Relationship between study hours and final grades
	•	Attendance patterns between students
	•	Visualizing the data using scatter plots and bar charts

Tools used:
	•	Python
	•	Pandas
	•	Matplotlib

Files in this project:
	•	student-performance.py
	•	students.csv
